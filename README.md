# 🎓 Student Management Portal

A modern and responsive **Student Management Web Application** built to simplify student record handling, course management, attendance tracking, and academic administration.

This project helps educational institutions manage student-related operations efficiently through an easy-to-use dashboard interface.

---

## 🚀 Features

* 👨‍🎓 Student Registration & Management
* 📚 Course Management
* 📝 Attendance Tracking
* 📊 Student Performance Records
* 🔍 Search & Filter Students
* 🔐 Authentication System
* 📱 Fully Responsive UI
* ⚡ Fast and User-Friendly Interface

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/patelsaba919-design/Student-managment-portal.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd Student-managment-portal
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start the Development Server

```bash
npm start
```
